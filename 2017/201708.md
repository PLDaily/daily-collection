## Difficulties & Bug

### 一

> IE下input['type=file']监听change事件判断是否有图片上传，上传成功后设置$(this).val('');会再次触发change事件。

> 在change事件中判断$(this).val()是否为空，为空则return false;

```javascript
if($(this).val) === '') {
  return false; 
}
```
